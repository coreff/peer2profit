### Usage
- Interactive
```shell
wget https://raw.githubusercontent.com/coreff/peer2profit/main/peer2fly.sh
chmod +x peer2fly.sh
./peer2fly.sh
```
- One Click Script
```shell
wget https://raw.githubusercontent.com/coreff/peer2profit/main/peer2fly.sh
chmod +x peer2fly.sh
./peer2fly.sh --email "Your email" --number "Container numbers"
```
